## Quick Start

### Introduction

An ui component base on react.
Color design refer to [Ant-Design](https://ant.design/docs/spec/colors).

### Installtation

```shell
npm install boy-ui --save
# or
yarn add boy-ui
# or
pnpm add boy-ui
```

### Usage

```ts
import { Button } from 'boy-ui';

export default function Demo() {
  return <Button text="Demo" />;
}
```

| --------------- | --------------- |
